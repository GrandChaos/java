Задание 1.
Написать два способа перемножения натуральных чисел (BigInteger):
- без использования потоков;
- с использованием потоков.
Количество потоков во втором варианте задавать переменной.
Сравнить результаты первого варианта и второго варианта (при разном количестве потоков).
Например, можно взять перемножение 100 000 первых натуральных чисел.

Задание 2 - Эстафета.
Есть N команд, в каждой из которых M бегунов.
Команды одновременно начинают бежать.
Каждый i+1 бегун может начать бежать только после того, как i бегун закончит бежать:
I.........I.........|
.....I....I.........|
.........II.........|
.........I.....I....|
.........I.........I|
Требуется выводить следующую информацию:
- в какое время начинает бежать команда;
- в какой время заканчивает бежать каждый бегун;
- в какое время заканчивает бежать команда;
- место, которое заняла каждая команда.
Для определения количества времени, требуемого для каждого бегуна, можно использовать класс java.util.Random:
- java.time.Random#nextInt().
Для определения времени можно использовать java.time.Duration и java.time.LocalDateTime:
- java.time.LocalDateTime#now();
- java.time.Duration#between(Temporal, Temporal).
